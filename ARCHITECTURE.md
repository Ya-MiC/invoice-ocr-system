# 系统架构

## 整体架构

```
┌─────────────────────────────────────────────────────────────┐
│                     FastAPI 服务层                           │
│  /health  /status  /process                                 │
├─────────────────────────────────────────────────────────────┤
│                   APScheduler 调度层                         │
│  定时扫描 → 任务队列 → 重试机制                              │
├─────────────────────────────────────────────────────────────┤
│                     业务处理层                               │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐         │
│  │ 文件处理器  │  │ OCR引擎封装 │  │ 发票解析器  │         │
│  │ (去重/锁)   │  │ (RapidOCR)  │  │ (字段提取)  │         │
│  └─────────────┘  └─────────────┘  └─────────────┘         │
├─────────────────────────────────────────────────────────────┤
│                   Rust 高性能层                              │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐         │
│  │ 文件监听    │  │ 图像预处理  │  │ 哈希计算    │         │
│  │ (notify)    │  │ (resize)    │  │ (SHA256)    │         │
│  └─────────────┘  └─────────────┘  └─────────────┘         │
├─────────────────────────────────────────────────────────────┤
│                     存储层                                   │
│  CSV输出  │  SQLite数据库  │  日志归档                       │
└─────────────────────────────────────────────────────────────┘
```

## Python vs Rust 职责划分

### Python 负责
- FastAPI HTTP服务
- APScheduler定时任务调度
- 业务逻辑编排
- 配置解析
- 数据库写入
- 错误处理与重试

### Rust 负责（通过PyO3暴露）
- 高性能文件监听
- 图像预处理（resize、去噪、二值化）
- 文件哈希计算（SHA256去重）
- 文件系统操作

## 数据流

```
input/  →  扫描  →  去重  →  锁定  →  OCR  →  解析  →  输出  →  processed/
                    ↓                        ↓
                  哈希库                  CSV/SQLite
```

## 关键模块

### main.py
- FastAPI应用入口
- 健康检查端点
- 任务状态查询
- 启动事件处理

### scheduler.py
- APScheduler配置
- 定时扫描任务
- 任务队列管理
- 重试调度

### ocr_engine.py
- RapidOCR封装
- 图像预处理调用
- OCR结果处理

### file_processor.py
- 文件去重（SHA256）
- 文件锁机制
- 目录扫描
- 文件移动

### invoice_parser.py
- 发票字段正则匹配
- 结构化数据提取
- 字段验证

### output_writer.py
- CSV写入
- SQLite写入
- 输出格式切换

### monitor.py
- CPU/内存/GPU监控
- 阈值告警
- 资源日志记录

## 错误处理

```
处理失败 → 记录错误 → 指数退避 → 重试 → 成功/归档
                         ↓
                    超过最大次数 → 移动到failed/
```
